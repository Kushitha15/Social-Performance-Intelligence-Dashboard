# Social-Performance-Intelligence-Dashboard
An interactive MS Excel dashboard designed to analyze social media performance, track key engagement metrics, and provide actionable insights to optimize content strategy.
This project focuses on analyzing social media data to measure content effectiveness and audience engagement. The dashboard enables users to monitor performance metrics and identify patterns to improve social media marketing strategies.

**The dashboard was built using the following tools and technologies:**
Microsoft Excel – Main platform for dashboard creation
Power Query – Data cleaning, transformation, and ETL process
Pivot Tables & Pivot Charts – Data aggregation and analysis
Excel Formulas – KPI calculations (Engagement Rate, CTR, etc.)
Data Visualization – Interactive charts and dashboard design

Data Source:Excel Datasets for Practice – Free Downloadable Samples

**Business Problem**
Social media teams often struggle to measure content effectiveness and identify what drives engagement and reach.

**Goal of the Dashboard**
Track key performance metrics (Engagement Rate, CTR, Reach)
Identify top-performing content
Analyze hashtag effectiveness
Optimize posting strategy based on data

This dashboard helps businesses and marketers improve their social media performance by providing clear insights into engagement trends, content effectiveness, and audience behavior.
